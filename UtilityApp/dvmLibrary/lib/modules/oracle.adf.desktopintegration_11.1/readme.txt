===============================
adfdi-excel-addin-installer.exe
===============================

The Windows executable located in this directory called 'adfdi-excel-addin-installer.exe' is
the full installer for the Oracle ADF Desktop Integration Add-In for Excel. This installer may
be used by both developers and end users.

This client installer is intended to be used with the corresponding shared
library oracle.adf.desktopintegration.war located in this same directory. They are all
matched at the same version and build.

For more information on this product, please visit the following link:

http://www.oracle.com/technetwork/developer-tools/adf/overview/index-085534.html
